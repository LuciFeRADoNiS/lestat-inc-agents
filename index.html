03

Cross-Bot Routing

Atlas, Hephaestus, Mnemosyne adapter'larını sırayla çağırır, sonuçları sentezler.

Tek soruda: atlas.deadlines + hephaestus.todos + mnemosyne.health. Paralel sorgu, tek özet cevap. Yetkin olmadığı yerde "kaynağına yönlendir" der.
04

Fuzzy Match

Yanlış yazılan isim, eksik tarih, kısaltma — toleranslı eşleştirme.

"Daimler" / "Daimlər" / "DTAG" → tek entity. Tarih: "salı" → bu hafta salı. Kişi: kısmi soyad → kontak listesi cross-check. Levenshtein + semantic embedding hibrit.
05

Adapter Views

3 bot DB'sine read-only erişim — tek elden tüm sistem görünür.

atlas: brain.db (deadlines, knowledge_pool, reminders, meeting_summaries) · hephaestus: group_brain.db (todos, decisions, daily_summaries, messages, api_usage) · mnemosyne: HTTP health endpoint (Phase 5'te full RPC).
06

Outbox Write

Doğrudan dosya değiştirmez — outbox kuyruğuna yazar, Cowork sync eder.

obsidian-outbox-sync task'ı 15dk'da bir kuyruğu Obsidian vault'a append eder. Çakışma yok, audit trail var, rollback mümkün. Phase 4 yeniliği.
